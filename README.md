# Anubhab-Portfolio

My personal portfolio website showcasing my journey as an aspiring AI/ML Engineer and DSA enthusiast.

## Overview

This is a modern, responsive portfolio website built to highlight my skills, projects, and passion for Data Structures & Algorithms and Machine Learning. I'm an Indian B.Tech student focused on landing a software/AI role in Mumbai after graduation.

## Tech Stack

- **HTML5** - Semantic markup and structure
- **CSS3** - Responsive design, animations, and modern styling
- **JavaScript (Vanilla)** - Navigation, smooth scrolling, and interactivity
- **GitHub Pages** - Deployment and hosting

No heavy frameworks required - pure, clean, hand-written code!

## Features

- **Fully Responsive** - Works seamlessly on mobile, tablet, and desktop
- **Sticky Navigation** - Easy access to all sections
- **Smooth Scrolling** - Elegant navigation experience
- **Modern Design** - Clean, minimal, and professional aesthetic
- **Project Showcase** - Display of DSA, ML, and personal projects
- **Timeline Section** - Visual representation of my learning journey
- **Resume Integration** - Downloadable resume in PDF format

## How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Anubhab-Roy-Chowdhury/Anubhab-Portfolio.git
   cd Anubhab-Portfolio
   ```

2. **Open in your browser:**
   - Simply open `index.html` in your favorite browser
   - Or use a local server for better experience:
     ```bash
     # Using Python 3
     python -m http.server 8000
     # Then navigate to http://localhost:8000
     ```

3. **Customize the content:**
   - Edit `index.html` to add your information
   - Update `styles.css` for custom styling
   - Modify `script.js` for additional functionality

## How to Deploy / Update on GitHub Pages

1. **Initial Setup:**
   - Go to repository Settings → Pages
   - Select "Deploy from a branch"
   - Choose the `main` branch and root folder
   - Save

2. **Deploy Your Site:**
   - Your portfolio will be live at: `https://Anubhab-Roy-Chowdhury.github.io/Anubhab-Portfolio`
   - Any commits to `main` will automatically update your live site

3. **Updating Content:**
   ```bash
   # Make your changes locally
   git add .
   git commit -m "Update portfolio content"
   git push origin main
   # Changes go live automatically!
   ```

## Customization Guide

### Edit Your Information
1. Open `index.html`
2. Look for placeholders marked with `TODO:`
3. Replace:
   - Your name
   - Email address
   - GitHub and LinkedIn links
   - Project descriptions and links
   - Skills and technologies

### Modify Styling
- Edit `styles.css` to change colors, fonts, and layout
- All styles are clearly commented for easy customization

### Add More Sections
- Add new sections in `index.html`
- Style them in `styles.css`
- Update navigation in `script.js` if needed
